MvcPluggedIn

MvcPluggedIn extends the ASP.NET MVC/Web API framework with a dynamic plug-and-play system for developing new areas/modules. The framework incorporates aspects of Entity Framework Code First, Ninject, ELMAH, and the ArcGIS REST API.

Download .zip  Download .tar.gz  View on GitHub

Why?

The MvcPluggedIn framework allows development of extensible ASP.NET MVC applications using modules (essentially MVC areas) that can be dropped into a specified folder for consumption by the host application. Modules can be created alongside the host application or separately as their own MVC application.
